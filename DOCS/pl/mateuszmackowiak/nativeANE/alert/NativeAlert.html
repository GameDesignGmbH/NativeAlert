Classpublic class NativeAlert
InheritanceNativeAlert flash.events.EventDispatcher

Simple NativeAlert extension that allows you to Open device specific alerts and recieve information about what button the user pressed to close the alert. Dispatches NativeAlertEvent

cancelable:Boolean (default = true) — - on back button or outside relese closes with index -1 (only Android)
